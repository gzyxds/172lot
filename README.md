# VitePress 教程 🔥

有着 `vuepress` 的搭建经验，上手还算 OK 😃

官方文档目前还没有中文，于是自己尝试搭建了一下 🤗

**访问** 👀

- VitePress 命令 安装依赖 ：
  ```
    - pnpm（建议使用 `nrm` 切换到淘宝源） ： `pnpm install`
    - npm ： `npm install`
    - yarn ： `yarn install`
  ```
**启动** 💥
  ```
1. npm ： `npm run dev` 、 `npm run docs:dev`
2. pnpm ： `pnpm dev`
3. npm ： `npm run dev`
4. yarn ： `yarn dev`
5. Makefile ： `make dev`
  ```
**编译构建** 🛠️
  ```
1. `pnpm` ： `pnpm run docs:build` 、 `pnpm run build`
2. npm ： `npm run build`
3. yarn ： `yarn run build`
4. Makefile ： `make dist`
  ```
**预览打包** 📦
  ```
pnpm ： `preview`
  ```
